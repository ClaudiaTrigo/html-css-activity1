Glossari d'intel·ligències artificials musicals

Generació musical

AIVA
Què fa: composició automàtica de música orquestral i bandes sonores.
Usos: cinema, videojocs, anuncis.
Lloc web
Soundraw
Què fa: plataforma per crear música de fons a mida.
Usos: youtubers, podcasters, creadors de contingut.
Lloc web
Mubert
Què fa: estríming infinit de música generada.
Usos: ambients sonors, productivitat, discjòqueis.
Lloc web

Veus i cant

Synthesizer V
Què fa: motor de veus sintètiques ultrarealistes per a cant.
Usos: producció musical professional.
Lloc web
Voicify AI
Què fa: crea versions de cançons utilitzant veus clonades.
Usos: entreteniment, mescles virals.
Lloc web
Uberduck
Què fa: genera veus parlades o cantades amb personatges i celebritats
Usos: versions humorístiques, contingut viral
Lloc web

Producció i mescla

Endlesss
Què fa: plataforma col·laborativa d'improvisació musical.
Usos: jam sessions, cocreació.
Lloc web
Magenta Studio (Google)
Què fa: plugins per a Ableton Live que generen melodies i grooves.
Usos: compositors electrònics, productors.
Lloc web
LANDR
Què fa: masterització automàtica en línia.
Usos: músics independents, segells petits.
Lloc web

Manipulació d’àudio

StemRoller
Què fa: separa pistes d'àudio (veu, bateria, baix, etc.).
Usos: discjòqueis, productors de remescles.
Lloc web
LALAL.AI
Què fa: extracció de veus i instruments amb alta precisió
Usos: versions, karaoke, mescles.
Lloc web
Audioshake
Què fa: eina professional de separació de pistes.
Usos: indústria musical, remescla avançada.
Lloc web

Lletres i creativitat textual

These Lyrics Do Not Exist
Què fa: genera lletres de cançons en diversos gèneres
Usos: inspiració ràpida per a compositors.
Lloc web
Bored Humans Lyrics Generator
Què fa: crea lletres originals utilitzant models simples.
Usos: jocs creatius, maquetes experimentals.
Lloc web
ChatGPT (amb prompts musicals)
Què fa: escriu lletres adaptades a estils concrets.
Usos: composició, pluja d'idees creativa.
Lloc web
